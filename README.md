
# Sharpi Support Automation — POC Visual Final

POC visual alinhado à solução arquitetada no GitHub.

## O que demonstra
- visão geral da solução em 3 fases
- entrada do incidente via WhatsApp
- processo de triagem acontecendo
- antes e depois da triagem
- contexto técnico sugerido
- ticket estruturado
- alerta interno
- retorno ao cliente
- fase 2 com aprendizado
- fase 3 com artigo de autoatendimento
- histórico de incidentes processados

## Como rodar

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
